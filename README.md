# Environment variables
```shell
GOERLI_RPC_URL=<url>
GOERLI_CONTRACT_ADDRESS=<address>

UNIQUERC_RPC_URL=<url>
UNIQUERC_CONTRACT_ADDRESS=<address>

METAMASK_PRIVATE_KEY=<private key for deploy contract>
```

# Deploy

```shell
npm run deploy:goerli
```

```shell
npm run deploy:unique
```

# Compile

```shell
npm run compile
```

# Test

```shell
npm run test
```
